# SWOT 分析

> 狀態標籤：推估

## Strengths｜優勢

1. 恐龍主題辨識度高
2. 園區面積約 8000 坪，空間條件佳
3. 2026 升級後具備 12 項遊樂設施
4. 彰化與中部大型親子樂園供給相對不足

## Weaknesses｜劣勢

1. 收費模式爭議大
2. 品牌曝光不足
3. 缺乏角色、世界觀與任務型 IP
4. 基礎服務體驗仍有待驗證與補強

## Opportunities｜機會

1. 親子旅遊市場持續成長
2. 短影音平台能快速放大景點聲量
3. 恐龍題材適合打卡與 UGC 傳播
4. 可串接彰化、鹿港、員林等周邊旅遊路線

## Threats｜威脅

1. 親子農場與休閒園區類型競爭者多
2. 大型遊樂園在整日型需求上更有優勢
3. 若負評持續累積，會壓縮重新開幕紅利

## 顧問結論

百果山的外部機會大於內部劣勢，但要吃到這波機會，前提是品牌必須先把「恐龍主題 + 親子體驗 + 可分享內容」重新組合成一致的市場語言。
