# 競爭景點分析

> 狀態標籤：推估

## 競爭類型

百果山的主要競爭者，不完全是大型遊樂園，而是以下兩類：

1. 親子農場與休閒園區
2. 可拍照、可放電的半日型親子景點

## 代表性競爭景點

### 綠世界農場｜新竹

- 生態與戶外體驗強
- 親子市場成熟

### 飛牛牧場｜苗栗

- 親子旅遊品牌成熟
- 農場與互動體驗具知名度

### 清境農場｜南投

- 山區旅遊吸引力強
- 一日與過夜市場都能承接

### 魔菇部落休閒農場｜彰化

- 區域內競品
- 在地親子市場重疊度高

### 麗寶樂園、劍湖山等大型遊樂園

- 屬於整日型競品
- 在「值不值得玩一天」的決策上會被比較

## 百果山的競爭優勢

- 恐龍主題更鮮明
- 森林景觀與拍照性強
- 半日行程門檻低
- 位於彰化，具在地親子出遊便利性

## 顧問觀點

百果山不應該跟大型遊樂園硬碰硬，而要搶占「中部最好拍、最好帶小孩放電、最好做半日行程」的心智位置。
