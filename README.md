# 百果山探索樂園顧問資料庫

這個專案整理了「百果山探索樂園」目前可用的顧問分析資料，包含園區歷史、設施、票價、營運模式、SWOT、行銷策略、競爭環境、最終顧問診斷，以及一份專門標記不確定資訊的待驗證清單。

## 專案內容

- `index.html`：靜態瀏覽首頁
- `styles.css`：首頁樣式
- `app.js`：首頁資料渲染
- `docs/`：Markdown 知識庫
- `meeting-reports/`：會議報告與模板
- `monthly-reports/`：月報與模板
- `quarterly-reports/`：季度顧問報告
- `client-report/`：業主版入口頁

直接用瀏覽器開啟 `index.html` 即可瀏覽專案首頁，Markdown 文件可直接閱讀或匯入 Notion、Obsidian、知識庫系統。
